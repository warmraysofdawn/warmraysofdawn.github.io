Интуитивное управление без навыков программирования

Быстрый старт

1

Windows

Установите Python версии 3.6 или старше. Скачайте архив с припиской WIN из Releases. Распакуйте и запустите start.bat

2

Linux/MacOS

Установите Python версии 3.6 или старше. Скачайте архив с припиской Linux из Releases. Распакуйте и запустите start.sh

3

Универсальный метод

Клонируйте репозиторий: git clone https://github.com/warmraysofdawn/SoRUPP
Установите зависимости: pip install -r requirements.txt
Запустите: python launch.py

После запуска откроется GUI в вашем браузере по адресу: http://127.0.0.1:5000

Часто задаваемые вопросы

❓ Нужны ли навыки программирования?

Нет! Интерфейс интуитивно понятен и не требует программирования.

❓ Можно ли изменить дизайн после экспорта?

Да, вы можете редактировать CSS файлы или заново запустить фреймворк с другим шаблоном.

❓ Поддерживаются ли пользовательские домены?

Да, экспортированный сайт можно разместить на любом домене.

❓ Как часто обновляется проект?

Мы регулярно выпускаем обновления. Вы можете стимулировать разработку, отмечая репозиторий звёздочкой.
